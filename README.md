# Student Sorting App

Консольное приложение для сортировки списка студентов.

## Запуск
1. Поместите `input.csv` в корень проекта.

2. Выберите способ запуска:

- **Maven**:
mvn clean package
java -jar target/student-sort-app-*.jar

- **IDE**: запустите Main.main().

- **Командная строка**:
javac -d out src/main/java/com/example/studentsort/**/*.java
java -cp out com.example.studentsort.Main

В меню выберите:
1 — ввести данные (источник: 1‑консоль, 2‑файл, 3‑случайные);
2 — сортировка по группе;
3 — сортировка по баллу;
4 — сортировка по зачётке;
0 — выход.

## Формат input.csv

101,4.5,ZK123456
102,3.8,ST654321

- Поля: группа,балл,зачётка (через запятую).
- Без заголовков и пустых строк.
- 1–100 записей.

## Валидация
- Группа: 1–999 (целое).
- Балл: 2.0–5.0 (1 знак после запятой).
- Зачётка: ≥3 символа, не пустая.

## Тех. требования
- Java 17+
- Maven 3.8+ (опционально)

## Структура
- model/ — данные;
- strategy/ — стратегии;
- service/ — сервисы;
- StuOrgApp.java — UI;
- Main.java — точка входа.

# Student Sorting App

Консольное приложение для сортировки списка студентов.

## Запуск
1. Поместите `input.csv` в корень проекта.

2. Выберите способ запуска:

- **Maven**:
mvn clean package
java -jar target/Student-organizer-*.jar

- **IDE**: запустите Main.main().

- **Командная строка**:
javac -d out src/main/java/**/*.java
java -cp out Main

В меню выберите:
- 1 — ввести данные (источник: 1‑консоль, 2‑файл, 3‑случайные);
- 2 — сортировка;
- 0 — выход.

## Формат input.csv

101,4.5,ZK123456
102,3.8,ST654321

- Поля: группа,балл,зачётка (через запятую).
- Без заголовков и пустых строк.
- 1–100 записей.

## Валидация
- Группа: 1–999 (целое).
- Балл: 2.0–5.0 (1 знак после запятой).
- Зачётка: 2 заглавных латинских буквы + 6 цифр.

## Тех. требования
- Java 17+
- Maven 3.8+ (опционально)

## Структура
- model/ — данные;
- input/ — ввод данных;
- sorting/ — сортировка;
- validation/ - влидация входящих данных;
- output/ - запись вывода в файл
- counter/ - подсчет количества студентов заданной группы
- collection/ - кастомная коллекция, заполнение через stream
- StuOrgApp.java — приложение для сортировки студентов;
- Main.java — точка входа.
